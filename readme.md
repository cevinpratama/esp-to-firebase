#  Project ESP ke Firebase (ESP32 & ESP8266)

Repository ini berisi contoh kode dasar untuk menghubungkan mikrokontroler **ESP32** dan **ESP8266** ke **Firebase Realtime Database (RTDB)**.

Project ini dibuat menggunakan **Visual Studio Code** dengan ekstensi **PlatformIO**.

---

##  Fitur

* Menghubungkan ESP32/ESP8266 ke jaringan WiFi.
* Melakukan autentikasi dan koneksi ke Firebase RTDB.
* Mengirim data (`setString`) ke path tertentu di database.
* Struktur project tunggal yang mendukung dua arsitektur board berbeda (ESP32 dan ESP8266).

---

##  Kebutuhan (Prerequisites)

Sebelum memulai, pastikan Anda memiliki:

1.  **Visual Studio Code**: Editor kode.
2.  **Ekstensi PlatformIO IDE**: Instal dari dalam VS Code Marketplace.
3.  **Board ESP32 atau ESP8266**: (Contoh: NodeMCU, Wemos D1 Mini, ESP32 DevKitC).
4.  **Akun Google Firebase**: Untuk membuat database.
5.  **Library Firebase ESP Client**: [Unduh dari link ini](https://drive.google.com/drive/folders/1Vx7c18sCkBwMoATT5D4FFhKHo6uTiWfI?usp=sharing)
